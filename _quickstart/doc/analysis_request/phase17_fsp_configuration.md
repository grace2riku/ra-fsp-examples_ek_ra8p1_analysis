# Phase 17: FSP設定解析

## カテゴリ
構成・設定編

## 概要
EK-RA8P1 _quickstartサンプルプログラムにおいて、FSP（Flexible Software Package）コンフィギュレータで設定されている内容を包括的に解析する。プロジェクト新規作成時のデフォルト値から変更されているFSP設定を特定し、一覧表として整理する。

## 背景・動機
_quickstartサンプルプログラムをベースに新規プロジェクトを作成する場合、FSPの設定とコード生成が極めて重要となる。どのような設定がデフォルトから変更されているかを把握することで、新規プロジェクト作成時に必要な設定作業を明確にする。

## 解析要求事項

### 1. FSP設定変更一覧表
以下の項目を含む一覧表を作成すること：

| 項目 | 説明 |
|------|------|
| 設定メニューの場所 | FSPコンフィギュレータ上のメニュー階層パス |
| 設定変更内容 | デフォルト値から変更された設定値 |
| 設定変更理由 | なぜその設定が必要か |
| FSP出力コード | FSPが生成するソースコードファイルと該当箇所 |
| FSPを利用しているところ | アプリケーションコード側で生成コードを利用している箇所 |

### 2. 解析対象カテゴリ

#### 2.1 プロジェクト基本設定
- RTOS選択（FreeRTOS）
- ツールチェーン設定
- ボード選択

#### 2.2 BSP設定（RA Common）
- メインスタックサイズ
- ヒープサイズ
- パラメータチェック
- その他の基本設定

#### 2.3 BSP設定（RA8P1 Family）
- SDRAM設定
- キャッシュ設定
- セキュリティ設定

#### 2.4 クロック設定
- PLL1/PLL2の設定
- 各クロックドメインの分周比
- 周辺クロックの有効/無効

#### 2.5 スレッド設定
- FreeRTOSスレッド一覧
- スタックサイズ、優先度
- RTOS同期オブジェクト

#### 2.6 モジュール（ドライバ）設定
- 追加された各FSPモジュール
- 各モジュールのプロパティ設定

#### 2.7 FreeRTOS設定
- ヒープサイズ
- タスク設定
- 機能の有効/無効

#### 2.8 ピン設定
- 各ペリフェラルのピン割り当て

## 期待する成果物

1. **FSP設定変更一覧表**: カテゴリ別に整理された設定変更の完全リスト
2. **クロックツリー概要**: 設定されたクロック構成の概要図
3. **モジュール構成図**: 各スレッドに紐づくモジュールの関係
4. **生成コードマッピング**: FSP設定と生成コード・利用コードの対応関係
